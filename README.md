# take

* [Homepage](https://osandriets.github.io/take/build/) - Demo take

## Developers

Clone the take repository.

```
git clone https://github.com/osandriets/take.git
cd take
```

Install the Node dependencies.

```
npm install
```

Install npm install bower

```
bower init
```

Run project

```
gulp
```

Navigate to http://localhost:9000
